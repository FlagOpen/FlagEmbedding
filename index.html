Besides the resources we provide here in this documentation, please visit our GitHub repo for more contents including:

  • Want to get familiar with BGE quickly? There are hands-on examples to run for embedder and reranker’s inference, evaluation, and finetuning.

  • Unfamiliar with some area, keywords or techniques of retrieval and RAG? We provide tutorials to teach you basic knowledge and coding tips.
